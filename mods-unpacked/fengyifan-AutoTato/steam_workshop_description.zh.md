[b][size=6]AutoTato 使用说明[/size][/b]

[b][size=5]这是什么？[/size][/b]
AutoTato 是一个商店自动化模组。它能帮你自动购买物品、自动刷新商店，让你更专注于战斗本身。

[b][size=5]基本操作[/size][/b]

[b]AutoTato 按钮[/b]
进入商店后，标题栏会有一个「AutoTato & 刷新」按钮。
- 点击按钮：执行一次商店决策（自动买/锁/跳），然后尝试刷新一次商店。
- 按下快捷键（手柄B / 键盘R）：在物品卡片上打开规则配置弹窗，设定每个物品你要怎么处理。商店和箱子物品界面都支持。

[b]刷新按钮[/b]
游戏自带的刷新按钮用于刷新商店。自动化操作请通过 AutoTato 按钮触发。

[b]设置面板[/b]
在暂停菜单可以找到 AutoTato 的设置面板。主要设置项：
- 商店自动化：总开关。关闭时进商店不会自动处理，但仍可点击 AutoTato 按钮手动触发一轮。
- 急速模式：开启后自动化瞬间完成所有操作，不做动画延迟。
- 最低金币保留：买完物品后至少保留多少金币（防买光）。
- 物品价格上限：超过这个价格的物品不自动买。设为 0 表示不限制。
- 刷新金额上限：单次刷新超过这个价格就不自动刷。设为 0 表示不限制（只看金币够不够）。
- 自动开始下一关：自动化处理完后如果没法再刷新了，自动点「下一波」按钮进入战斗。

[b]物品规则设置[/b]
在商店或箱子物品界面，对着物品卡片按手柄B或键盘R，可以打开规则配置弹窗，同时设置该物品的两种行为：
- 商店行为：在商店里遇到这个物品时怎么处理（购买 / 拒绝 / 仅诅咒 / 锁定等诅咒 / 手动）。
- 箱子行为：开箱子开出这个物品时怎么处理（拿取 / 拒绝 / 仅诅咒 / 手动）。

商店行为可选：
- 购买：遇到就自动买
- 拒绝：不买这个物品
- 仅诅咒：只买诅咒版本
- 锁定等诅咒：非诅咒版本先锁住，等诅咒版本再买（鱼钩神力）
- 手动：不自动处理，留给你自己决定（这是默认行为）

箱子行为可选：
- 拿取：开出就自动拿
- 拒绝：开出就自动丢弃
- 仅诅咒：只拿诅咒版本，非诅咒自动丢弃
- 手动：不自动处理，留给你自己决定（这是默认行为）

[b]武器规则设置[/b]
武器的规则设置与物品类似，额外支持两个选项：
- 跟随类别：和该武器所属类别（如「枪械」）的规则一致，如果任一类别为手动，则物品被判定为手动。
- 等级下限：设置最低武器等级，低于这个等级的武器自动跳过。

[b][size=5]特殊行为说明[/size][/b]
以下是一些你可能不容易注意到的行为逻辑。

[b]手动锁定商品与自动化的交互[/b]
如果你在商店里手动锁定了一个商品（防止被刷新刷掉），而这个商品的规则恰好是「手动」，自动化会把它当作「跳过」处理。
这意味着：你手动锁定的商品不会触发「出现手动，停止自动化」。自动化会继续处理其他商品，而锁定的那个会保留到下一波。

[b]未设置规则的物品默认不动[/b]
如果你从没给某个物品设过规则，它默认是「手动」—— 自动化不会动它，留给你自己决定。
未设置规则的武器也一样，默认「手动」，不会自动买。

[b]全部锁定后不会刷新[/b]
如果你把商店里所有商品都锁定了（4个全锁），自动化会检测到刷新没效果，不会去刷新。

[b]关于禁用物品（ban）[/b]
在 AutoTato 下，原生的禁用物品功能不可用：手柄B / 键盘R 用于打开物品规则配置弹窗，没有单独的禁用物品操作。所有物品的处理方式都通过规则弹窗来配置。如果你在玩禁用挑战模式，请通过规则弹窗的「拒绝」选项来达到类似效果。

[b]规则弹窗的关闭方式[/b]
规则弹窗打开后，可以用以下方式关闭：
- 按ESC（键盘）或手柄B：关闭弹窗，不会触发暂停菜单。
- 点击弹窗外的半透明遮罩：关闭弹窗。
- 点击「保存」或「取消」按钮：关闭弹窗（保存会应用规则，取消则不保存）。
注意：用手柄B打开的弹窗，第一次松开B不会关闭弹窗（防止按下开、松开关的抖动），需要再按一次B才会关闭。

[b]自动化关闭时按钮仍可用[/b]
即使关掉了「商店自动化」开关，点击 AutoTato 按钮仍然会执行一轮决策并尝试刷新。

[b]自动刷新会自己停下来[/b]
开启自动化后，遇到以下任一情况会自动停止刷新：
1. 出现「手动」商品（需要你决定）
2. 所有商品都买不起
3. 金币不够付刷新费
4. 单次刷新价格超过你设的上限

不会无限刷新花光你的金币。

[b]刷新金额上限是单次价格[/b]
「刷新金额上限」这个设置控制的是单次刷新价格，不是累计总额。超过这个价格就不自动刷。设为0表示不限制。
