[h1]AutoTato 使用说明[/h1]

[h2]这是什么？[/h2]
AutoTato 是一个自动化模组。自动购买商店物品、自动刷新、自动选择升级，让你更专注于战斗。

[h2]基本操作[/h2]

[b]商店 AutoTato 按钮[/b]
进入商店后，标题栏有「AutoTato & 刷新」按钮。
[list]
[*]点击：执行一次商店决策（买/锁/跳），然后刷新一次。
[*]手柄B / 键盘R：在物品卡片上打开规则弹窗。商店和箱子界面都支持。
[/list]

[b]升级 AutoTato 按钮[/b]
升级界面（4 选 1）刷新按钮下方有「AutoTato」按钮。
[list]
[*]点击：执行一次升级决策（按策略选出最优项并选下）。
[*]手柄X / 键盘E：同效。与箱子 AutoTato 按钮共用快捷键。
[*]开启「升级自动化」后，进入界面自动开始决策。
[/list]

[b]刷新按钮[/b]
原版刷新按钮只刷新。自动化请通过 AutoTato 按钮触发。

[b]设置面板[/b]
暂停菜单中可找到。主要设置：
[list]
[*]商店自动化：总开关。关闭后进商店不自动处理，但按钮仍可手动触发。
[*]升级自动化：总开关。关闭后进升级界面不自动选，但按钮（或 E 键）仍可手动触发。
[*]急速模式：瞬间完成所有操作，无动画延迟。
[*]最低金币保留：买完后至少保留多少金币（防买光）。
[*]物品价格上限：超过此价格不自动买。0 = 不限。
[*]刷新金额上限：单次刷新超过此价格就不自动刷。0 = 不限（只看金币够不够）。
[*]自动开始下一关：无法再刷新时自动点「下一波」进入战斗。
[/list]

[b]升级策略[/b]
[list]
[*]受阈值影响：当升级项涉及的所有相关属性都触达阈值上限时，过滤该选项。
[*]最低等级：低于此等级（普通/精良/史诗/传说）的选项直接跳过。
[*]品质优先：优先选高等级（稀有度）选项。
[*]禁止属性：含这些属性的选项被过滤。
[*]卡住时忽略禁止列表：全部被过滤时回退到不过滤排序（选品质最优）。
[*]属性优先级：在最高等级候选中，按你排的属性顺序优先选择。
[/list]

[b]物品规则[/b]
按手柄B / 键盘R 打开规则弹窗，同时设置两种行为：
[list]
[*]商店行为：购买 / 拒绝 / 仅诅咒 / 锁定等诅咒 / 手动。
[*]箱子行为：拿取 / 拒绝 / 仅诅咒 / 手动。
[/list]
商店选项：购买（遇到就买）、拒绝（不买）、仅诅咒（只买诅咒版）、锁定等诅咒（非诅咒先锁，等诅咒再买，鱼钩神力）、手动（留给你，默认）。
箱子选项：拿取（开出就拿）、拒绝（开出就丢）、仅诅咒（只拿诅咒版，其余丢弃）、手动（默认）。

[b]武器规则[/b]
同物品，额外支持：
[list]
[*]跟随类别：跟随武器所属类别（如「枪械」）的规则。任一类别为手动则武器为手动。
[*]等级下限：低于此等级的武器自动跳过。
[/list]

[h2]特殊行为[/h2]

[b]手动锁定与自动化[/b]
手动锁定的商品若规则为「手动」，会被当作「跳过」--不会停止自动化。自动化继续处理其他商品，锁定的留到下一波。

[b]无规则 = 不动[/b]
未设规则的物品和武器默认「手动」，自动化不会动。

[b]全锁 = 不刷新[/b]
4 个商品全锁时，自动化检测到刷新无效，跳过刷新。

[b]关于禁用物品[/b]
原版禁用功能不可用：手柄B / 键盘R 用于打开规则弹窗。禁用挑战模式请用规则弹窗的「拒绝」达到类似效果。

[b]规则弹窗关闭方式[/b]
[list]
[*]ESC / 手柄B：关闭（不触发暂停菜单）。
[*]点击弹窗外遮罩：关闭。
[*]保存/取消按钮：关闭（保存应用规则，取消不保存）。
[/list]
注：手柄B 打开的弹窗，需再按一次B 才关闭（防抖动）。

[b]关闭自动化时按钮仍可用[/b]
即使关闭总开关，点击 AutoTato 按钮仍会执行一轮决策。

[b]升级关闭时：按一次进一步[/b]
「升级自动化」关闭时，每按一次按钮（或 E 键）执行一轮：有合格项就选下；没有则刷新一次后停下。不会连续自动刷新，每次推进需手动触发。

[b]升级开启时：刷新循环[/b]
「升级自动化」开启时，进入界面自动开始：
[olist]
[*]对 4 个候选做决策（过滤 + 排序）。
[*]有合格项 -> 选下，结束。
[*]无合格项 -> 刷新一次，重复。
[*]直到选中、金币不足、或刷新价格超限。
[/olist]

[b]卡住兜底[/b]
金币耗尽仍无合格候选时，若「卡住时忽略禁止列表」开启，自动选当前等级最高的可见项。否则交还手动。

[b]自动刷新会自己停[/b]
遇到以下任一情况停止：
[olist]
[*]出现「手动」商品（需你决定）。
[*]所有商品都买不起。
[*]金币不够付刷新费。
[*]单次刷新价格超限。
[/olist]
不会无限刷新花光金币。

[b]刷新金额上限 = 单次价格[/b]
此设置限制单次刷新价格，非累计总额。0 = 不限。
